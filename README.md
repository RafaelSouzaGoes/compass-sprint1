####Tema: Segurança de redes: Conheça as vulnerabilidades de servidores e clientes

1. O que é um ataque DDoS e como posso me proteger?
    <br/>

    * Normalmente utilizando botnets o hacker controla maquinas infectadas e inicia um ataque contra um serviço tornando praticamente impossível de ser acessado por usuários comuns fazendo as vezes o serviço web fique indisponível.
    <br/>

2. Por que o firewall é uma ferramenta muito importante de proteção?
    <br/>
    * Como o firewall seleciona os níveis de segurança das partes da rede onde são separadas por zonas, a interna chamada de zona 100 por ser totalmente segura, a externa zona 0 por ser totalmente insegura e a do servidor web que é chamada de zona 50 ou zona desmilitarizada (DMZ), com isso o firewall limitar o acesso de zonas menos segura para zonas mais seguras. Impedindo assim hackers de invadir com facilidade a rede interna da empresa ou residência.
